【簡介】
─────────────────────────────────────────────
專為中國醫藥大學學士後中醫學系入學考試生物科設計的離線練習工具。
題庫整合中國醫 105–114 學年度全部 500 題歷屆考古題,並依
Campbell Biology 教材結構分章節對照,額外補上 80 題章節觀念
補強題,合計 580 題完整題庫。

所有作答紀錄、錯題、筆記、模擬考歷史皆儲存於瀏覽器本地端
(localStorage),無需網路、無需註冊、無資料外傳。

【啟動方式】
─────────────────────────────────────────────
方法一(推薦):雙擊開啟
  1. 將檔案下載至電腦
  2. 雙擊「後中醫生物學練習器.html」
  3. 系統會用預設瀏覽器開啟(建議 Chrome、Edge、Safari)

方法二:拖曳開啟
  將檔案直接拖入瀏覽器視窗即可

方法三:行動裝置
  ・iPhone/iPad:用「檔案」app 開啟,或先存至 iCloud Drive
    再用 Safari 開啟
  ・Android:下載至手機後,用 Chrome 或檔案管理員開啟

行動裝置使用建議:
  iOS Safari → 點分享 → 加入主畫面,即可像 app 一樣使用


【七大功能模組】
─────────────────────────────────────────────

(1) 概覽
    顯示題庫總數、已作答題數、總正確率、錯題待複習、今日已做、
    模擬考次數等關鍵指標。提供四個快速入口按鈕。

(2) 每日練習
    可選 10 / 20 / 30 / 50 題,支援以下篩選:
    ・年度:全部 / 105–114 任一學年
    ・來源:考古+章節重點 / 僅考古 / 僅章節重點
    ・抽題策略:隨機 / 未做過優先 / 含已錯題
    
    作答後即時顯示:答案、章節、解析(若有)、可一鍵記筆記

(3) 章節練習
    依 Campbell 八大單元(實際 10 個章節分類)篩選練習,
    每章節最多抽 30 題,適合針對弱項集中加強。

(4) 模擬考試
    ・50 題單選
    ・80 分鐘倒數計時(超時自動交卷)
    ・計分規則完全比照中國醫:每題 2 分、答錯倒扣 0.7 分、
      未作答不計分
    ・可標記題目、自由跳題、即時統計已答/已標數
    ・交卷後顯示總分、各章節答對率長條圖、可逐題檢討

(5) 錯題複習
    自動收錄歷次答錯的題目。
    ・依章節篩選
    ・可挑戰所有錯題或隨機 10 題
    ・反覆做直到答對為止會從錯題庫消失

(6) 知識點筆記
    ・可從任一題目一鍵建立筆記(自動連結至題目)
    ・依章節分類管理
    ・可編輯、刪除、跳回原題目
    ・適合搭配艾賓豪斯遺忘曲線複習

(7) 學習報告
    ・基本統計:已答題數、正確率、模擬考次數/平均/最高分
    ・各章節答對率長條圖
    ・各年度答對率長條圖
    ・最近 10 次模擬考紀錄
    ・資料匯出(JSON 備份)
    ・資料匯入(換瀏覽器/裝置時可帶走)
    ・清除所有紀錄(雙重確認)


【計分規則(模擬考)】
─────────────────────────────────────────────
完全比照中國醫 111 學年起的計分標準:

  得分 = 答對題數 × 2 - 答錯題數 × 0.7
  滿分 100 分,負分時計為 0 分
  未作答不加分也不扣分

策略提醒:
  ・單純猜答(1/4 機率對)期望值:0.25×2 - 0.75×0.7 = -0.025 → 微負
  ・能刪去 2 個明顯錯誤選項時(1/2 機率對):
    0.5×2 - 0.5×0.7 = 0.65 → 應該答
  ・能刪去 3 個錯誤選項時:幾乎一定要答

【資料儲存與隱私】
─────────────────────────────────────────────
所有資料儲存於瀏覽器的 localStorage:

  bio_history       作答歷史
  bio_notes         筆記內容
  bio_mockHistory   模擬考歷史

特性:
  ✓ 完全本地端儲存,不傳送任何資料至外部
  ✓ 即使關閉瀏覽器、重開電腦,資料仍保留
  ✗ 切換瀏覽器(如 Chrome → Edge)資料不會跟過去
  ✗ 清除瀏覽器資料時,練習紀錄會一併消失!

備份建議:
  每週至「學習報告」頁底點「匯出資料」儲存 JSON 備份檔
  換裝置或瀏覽器時可用「匯入資料」還原進度


【使用建議】
─────────────────────────────────────────────
(1) 學習新章節
    → 章節練習 → 選該章 → 先做 8 題章節重點(觀念建立)
    → 看每題解析,把不懂的點記筆記

(2) 加強記憶
    → 每日練習 → 20-30 題 → 抽題策略「未做過優先」
    → 累積到一定量後改用「含已錯題」反覆鞏固

(3) 應試訓練
    → 兩週一次模擬考(50 題 80 分鐘)
    → 看交卷後的章節答對率長條圖,找出弱章節
    → 弱章節再回去做章節練習

(4) 考前衝刺
    → 錯題複習 → 把所有歷次錯題反覆做到全對
    → 翻閱知識點筆記查漏補缺
    → 最後一週做整年份模擬考(年度選 113、114)


【後續擴充】
─────────────────────────────────────────────
可請 Claude 補充:
  ・107–114 學年度未收錄的考古題
  ・特定章節再加 8-10 題重點題
  ・特定主題的深度題組(如:訊息傳遞、糖代謝、基因表現調控)
  ・英文重要名詞對照題(中翻英、英翻中)


【已知限制】
─────────────────────────────────────────────
1. 圖表類題目:原考題若涉及圖、譜系、流程圖,題幹會以文字描述,
   建議搭配考選部公布之原試題 PDF 對照閱讀效果最佳。

2. 章節分類:題目依關鍵字自動分類,少數跨章節題目可能歸類略有
   爭議,可依個人理解自行解讀。

3. 計分倒扣:模擬考完全模擬中國醫規則(111 學年起倒扣 0.7 分),
   若參加其他學校(如義守、慈濟、中國)考試,規則可能不同。
